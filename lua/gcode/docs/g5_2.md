# G5.2

NURBS block start - Opens Non-Uniform Rational B-Spline data block.

**Summary**
Experimental feature for complex curved surfaces. Control points defined between G5.2 and G5.3.

**Format**
`G5.2 <P<weight>> <L<order>> <X Y>`

**Example**
```
G0 X0 Y0
F10
G5.2 P1 L3
     X0 Y1 P1
     X2 Y2 P1
     X2 Y0 P1
     X0 Y0 P2
G5.3
```
