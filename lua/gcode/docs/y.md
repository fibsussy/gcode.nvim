# Y Axis

Y-axis coordinate for machining operations.

**Summary**
Specifies the Y-axis position in the current coordinate system. Used with G0, G1, G2, G3 and other positioning/motion commands.

**Format**
`Y<position>`

**Example**
```
G0 Y20 (rapid to Y=20)
G1 Y100 F200 (linear feed to Y=100)
```
