# G50

Coordinate system scaling / Maximum spindle speed (context-dependent).

**Summary**
Different meanings on different controls:
- Most controls: Coordinate system scaling (scale factor)
- Haas: Maximum spindle speed limit (with G50 S<rpm>)

**Format**
`G50 S<max_rpm>` or `G50 I<scale_x> J<scale_y> K<scale_z>`
