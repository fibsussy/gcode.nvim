# G91.1

Incremental arc distance mode - IJK offsets are relative to current position.

**Summary**
Default mode for most controllers. I, J, K words specify the offset from the current position to the arc center. Opposite of G90.1 (absolute arc distance mode).

**Format**
`G91.1`

**Example**
```
G91.1 (incremental arc offsets, default)
G2 X1 Y0 I0.5 J0.5 (I and J are offsets from current pos to center)
```
