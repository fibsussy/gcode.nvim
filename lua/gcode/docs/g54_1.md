# G54.1

Extended work coordinate system - Selects one of 48 additional WCS offsets.

**Summary**
Provides 48 extra fixture offsets beyond G54-G59. The P word selects which extended WCS to use (P1-P48). Offsets set via G10 L20 or touch-off.

**Format**
`G54.1 P<n>`

**Parameters**
- `P1`-`P48` — Extended WCS index

**Example**
```
G54.1 P10 (select extended WCS #10)
G54.1 P1  (select extended WCS #1)
```
